Один із видів селекторів

  • Класи Класи застосовують, коли необхідно визначити стиль для одного чи декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково чи повністю схожі елементи оформлення. У коді HTML ім'я класу визначається через атрибут "class."
  • Мультикласи До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. в цьому разі до елемента застосовується стиль, описаний в правилах для кожного класу. Оскільки під час додавання кількох класів вони можуть містити однакові стильові властивості, але з різними значеннями, то береться значення в класу, який описано в коді нижче.
  • Універсальний селектор Іноді потрібно встановити одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт чи накреслення тексту. в цьому випадку допоможе універсальний селектор, який відповідає будь-якому елементу веб-сторінки. в CSS3 універсальний селектор застосовується також з комбінації з простором імен. ns|* — всі елементи в просторі імен ns. *|* — всі елементи в просторі всіх імен. |* — всі елементи без явної вказівки простору імен. Для позначення універсального селектора застосовується символ зірочки (*). в деяких випадках вказувати універсальний селектор не обов'язково. Так, наприклад, записи *.class .class є ідентичними за своїм результатом.
  • #id Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю звернення до нього через скрипти Під час опису ідентифікатора спочатку вказується символ решітки (#), потім іде ім'я ідентифікатора. Воно має починатися з латинського символу може містити символ дефіса (-) підкреслення (_). Використання українських літер в іменах ідентифікатора неприпустимо. відміну від класів ідентифікатори мають бути унікальними, іншими словами, зустрічатися в коді документа тільки один раз. Звернення до ідентифікатора відбувається за допомогою атрибута id, значенням якого виступає ім'я ідентифікатора (id="header"). Символ решітки при цьому вже не вказується
